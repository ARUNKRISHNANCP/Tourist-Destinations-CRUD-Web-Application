# Tourist-Destinations-CRUD-Web-Application
Tourist Destinations CRUD Web Application is a Django and Django REST Framework-based web application that enables users to create, view, update, and delete tourist destination records, including destination name, weather, location, Google Maps link, description, and multiple images through a user-friendly interface

The Tourist Destinations CRUD Web Application is a web-based system developed using Django and Django REST Framework that allows users to manage information about tourist destinations efficiently. The application provides complete Create, Read, Update, and Delete (CRUD) operations for destination records, including details such as destination name, weather conditions, state, district, Google Maps location, description, and multiple images.

The system offers both a user-friendly frontend interface for managing destinations and a RESTful API for data access and integration. It helps users explore, add, update, and organize tourist destination information in a centralized platform.

Key Features:
Add new tourist destinations
View destination details and images
Update destination information
Delete destinations
Upload multiple destination images
Google Maps integration for location viewing
REST API support using Django REST Framework
Responsive and easy-to-use interface
Technologies Used:
Python
Django
Django REST Framework (DRF)
HTML
CSS
Bootstrap
SQLite/MySQL/PostgreSQL
Project Objective:

To provide a centralized platform for storing and managing tourist destination information while demonstrating the implementation of CRUD operations, REST APIs, file uploads, and frontend-backend integration using Django technologies.
